    User details

    User group First time user
    User journey User starts with google search, finds how to trade after Jan 2021. Brought through question flow to bring them to relevant, specific guide to their trade.

    Notes

    • Changed screen order so that transit comes after the start and end points.
    • Transit screen plays back choices from start and end point
    • Show search results of Orchids - link through to further details of each step.

    Content on start page:

    • settled on 'animals, plants and agricultural products' as this is used elsewhere in Defra for these types of commodities
    • high level list of in scope commodites
