Prototypes

Guidance Finder

  • Prototype v1

    User group - First time user - exisiting EU only trader, never needed to use a trade system (TRACES) before
    Prototype journey to guide the user to the appropriate information for them to be able to trade after Jan 1st 2020

    User journeys covered:

    • Importing plants - use "Orchids" as commodity type
    • Exporting fish - use "Salmon" as commodity type
    • Exporting horses - user "Horses" as commodity type

    Content on start page:

    • settled on 'animals, plants and agricultural products' as this is used elsewhere in Defra for these types of commodities
    • high level list of in scope commodites
    • advance warning of what user will be asked to do

  • Prototype v2

    User group - First time user - exisiting EU only trader, never needed to use a trade system (TRACES) before
    Improved prototype to show relevent action pages

    User journeys covered:

    • Importing plants - use "Orchids" as commodity type
    • Exporting fish - use "Salmon" as commodity type
    • Exporting horses - user "Horses" as commodity type

  • Prototype v3

    User group - First time user - exisiting EU only trader, never needed to use a trade system (TRACES) before
    Improved prototype to show relevent action pages

    Same user journeys as above - but have added transit screens.

  • Prototype v4

    User group - First time user - exisiting EU only trader, never needed to use a trade system (TRACES) before
    Improved prototype to show relevent action pages

    Same user journeys as above

    Updates:

    • Changed screen order so that transit comes after the start and end points.
    • Transit screen plays back choices from start and end point
    • Show search results of Orchids

    Content updates on start page:

    • more detailed list of in scope commodites
    • link for out of scope commodity
